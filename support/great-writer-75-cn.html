大文豪 75

75% 三模无线机械键盘套件

型号GW-75
配列75%（84键）
连接方式蓝牙1/2/3 · 2.4G · USB-C
版本Rev. 1.0 · 2025

欢迎使用大文豪75。本说明书涵盖连接设置、按键功能、RGB控制及电池管理，请妥善保存备用。

02

键位图

大文豪75采用紧凑型75%配列，保留完整F键行和方向键区，共84键。兼容标准MX轴座键帽套件。

03

连接方式

大文豪75支持蓝牙三设备、2.4G无线和USB-C有线三种模式。使用 FNQ/W/E 切换蓝牙频道，FNR 切换2.4G，FNT 切换USB有线。

01
蓝牙连接
按 FN+Q、FN+W 或 FN+E 分别进入蓝牙1、2、3配对模式。

长按约3秒，指示灯快速闪烁后，在设备端搜索"大文豪75"并连接。
02
2.4G 无线
插入随附USB接收器后，按 FN+R 切换到2.4G模式。

键盘与接收器自动配对，无需额外操作。
03
USB-C 有线
连接USB-C数据线后，按 FN+T 切换到有线模式。

有线模式同步为电池充电。
WIN / MAC 模式切换

按 FN+M 切换Windows和Mac键位映射。Mac模式下Win键映射为Command、Alt映射为Option，切换时RGB指示灯短暂闪烁确认。

全键无冲（NKRO）：按 FN+N 开关NKRO功能。建议游戏或快速输入时开启。注意：NKRO仅在USB-C有线模式下可用。

05

RGB 背光

用FN+方向键切换模式和亮度，FN+A/S/D/F/G/H精细调节色相、饱和度和动效速度。

切换模式
FN + ← / → 切换灯光效果模式
调节亮度
FN + ↑ / ↓ 增加或降低亮度
调节色相
FN + A（减）/ S（加）偏移颜色
调节饱和度
FN + D（减）/ F（加）调整色彩浓度
调节动效速度
FN + G（慢）/ H（快）
开关背光
FN + Enter 一键开启或关闭 RGB

提示：RGB设置会自动保存，断电后灯光模式和亮度保持不变。

06

电池与电源

查询电量按 FN + Space，RGB指示灯短暂闪烁显示电量：
● 绿色 = 60%以上  ● 黄色 = 20–60%  ● 红色 = 20%以下
充电连接USB-C数据线即可充电，充电期间键盘正常使用。充满后充电指示灯熄灭。
省电模式静置约3分钟后自动进入休眠，任意按键唤醒。
恢复出厂长按 FN + 左Shift + Backspace 约3秒恢复出厂，所有蓝牙配对信息将被清除。
