## 0. 一句话总览

**两个账号是互补的一对 · 不是两个独立博主**：

| | 不现实商店（品牌号） | AZ的不现实商店（个人号） |
|---|---|---|
| 关键词 | 视觉 / 产品 / 世界观 | 活人感 / 过程 / 手 |
| 目的 | 卖货 + 立品牌 | 信任 + 立人 |
| KPI | 粉丝数 + 转化 | 活人感的建立 不是涨粉 |
| 视角 | "我们设计了..." | "我..."  |

**两个号合在一起讲一个完整的故事**：一个有审美的人 在做一些不现实的事 一边做一边讲。

## 1. 这条内容发哪个号 · 决策树

```
内容类型？
│
├── 产品讲解（开箱/讲设计/讲工艺）
│     → 主发品牌号 + AZ号剪一版"设计师视角的1个细节"
│
├── 设计师视角思考（为什么这样做/纠结/灵感）
│     → AZ号
│
├── 过程记录（打样/试色/工厂/装店/直播）
│     → 浅一点（结果导向） = 品牌号
│     → 深一点（带挫败/思考） = AZ号
│
├── 行业观察 / 反思（像"图里有我"）
│     → AZ号 · 但稀缺使用 · 不做主食
│
├── 家里开店进度（战略主线）
│     → 主线品牌号 + AZ号吃幕后
│
└── 开箱评测别人产品
      → 都不做
```

## 2. 每个号的"硬要素"

### 品牌号 必须有
- ✅ 高质量产品图（调色精致 · 美学统一）
- ✅ 世界观铺陈（每颗按键有故事）
- ✅ "我们设计了..."的品牌视角
- ✅ 视觉锁定的封面美学

### AZ号 必须有（每条至少一个 "活人信号"）
- ✅ 脸
- ✅ 手的工作镜头
- ✅ 真实的说话声（含停顿/犹豫/口误）
- ✅ 生活痕迹（咖啡渍/凌乱桌面）
- ✅ 失败 / 返工 / 拍砸
- ✅ 累 / 笑 / 叹气 / 摇头

### AZ号 严格禁用
- ❌ 三脚架 / 稳定器
- ❌ 调色（除一个轻预设）
- ❌ 花字 / 动效
- ❌ BGM
- ❌ 精修 / 磨皮
- ❌ "我们"视角 — AZ号是"我"

## 3. 美学系统 · 一拍两标准

| | 品牌号产品图（工作） | AZ号内容（生活） |
|---|---|---|
| 标准 | 高质量调色 全力以赴 | 真实不修 活人感 |
| 设备 | 相机+灯+调色软件 | 手机 · 原片 |
| 时间投入 | 投入 · 值得 | 随手 · 不超过10分钟 |
| 三件套（固定） | 按产品需求 | ① 自然光 ② 一个轻预设 ③ 居中或45°俯拍 |

**美学逻辑**：你的美术调色能力**全部投到产品图**，AZ号反其道而行——**粗糙但统一才是活人感**。

## 4. 发布节奏

| 账号 | 节奏 | 内容类型 |
|---|---|---|
| 品牌号 | 日更 | 产品 / 世界观 / 视觉 |
| AZ号 | 高频轻量为主 + 每周1条完整 + 月度偶尔1条立场片 | 过程 / 手 / 真声 |

**AZ号 70/20/10内容结构**：
- 70% 主食 — 家里开店进度 / 键帽设计日常 / AI工作流（凸显人）/ 生活镜头 / 手作失败返工
- 20% 偶尔 — 深度反思/立场片（"图里有我"这种）/ 系列联动
- 10% 极少 — 行业观察 / 审美契合的合作

## 5. 平台分配

### 主阵地：小红书

| 平台 | 品牌号 | AZ号 | 时长 | 内容重点 |
|---|---|---|---|---|
| 小红书 | 主阵地 日更 | 主阵地 | 1-1.5分钟 | 完整 · 钩子+悬念+互动钩 |
| 抖音 | 拍一次剪一次 | 偶尔 | 1分钟以内 | 冲击 · 数字钩 · 节奏快 |
| B站 | 完整版 | 偶尔 | 2-3分钟 | 详细 · 简介可加自我介绍 |
| 视频号 | 周更 | 偶尔 | 1-1.5分钟 | 调性稳重 · 标签克制 |

### 多平台发布顺序（建议）
1. D-day：**小红书 + 视频号** 同时发
2. D+1：**抖音**（间隔避免算法判定搬运）
3. D+2 / D+3：**B站**（B站不在乎首发性）

## 6. 每个平台的载体硬规则（防超长/防错位）

| 载体 | 长度 | 写法 |
|---|---|---|
| 视频脚本（口播） | 按平台1-2分钟 | 口语化 · 有节奏 · 有情绪 |
| 小红书笔记正文 | **3-7行** | **不复述脚本** · 钩子+关键词+互动钩 |
| 抖音视频描述 | 1-2行 | 超短 · 直给 |
| B站视频简介 | 4-6行 | 可详细 · 末尾可加"——我是AZ..." |
| 视频号描述 | 1-2行 | 克制 |
| 封面文字 | 4-10字 | 极简 · 有悬念或反差 |
| 笔记标题 | 15-25字 | 搜索友好 + 情绪词 |
| 标签 | 平台不同 见下 |

### 标签数量
- 小红书：10-15个 · `#关键词`
- 抖音：8-10个 · `#话题`
- B站：8-12个 · 逗号分隔（不带#）
- 视频号：**5个以内** · 克制

## 7. 绝对不做（不论哪个号）

- ❌ 系统大课 / 知识付费大课（卖课犹豫是对的）
- ❌ 纯方法论教程（不做老师）
- ❌ 快速种草开箱（不做麓叔型博主）
- ❌ 鸡汤创业心得
- ❌ 装"成功创业者"
- ❌ 把脚本复制成笔记正文
- ❌ 在公开内容里承认大号流量天花板
- ❌ 中英文混排
- ❌ 任何让"不现实"变得不稀缺的事

## 9. 每条内容发布前的8-Pass评审

按 skill 文档执行（在 `.claude/skills/az-content-review/SKILL.md`）：

1. **战略** — 账号标签风险？时机对吗？关哪些门？
2. **身份** — 这条为什么是AZ讲？用了AZ的独家弹药吗？
3. **三必问** — 学到什么？马上做什么？跟AZ什么关系？
4. **结构** — 前3秒钩子？前30%痛点切入？后20%互动钩？
5. **语气** — 松弛？我当时vs你应该？自嘲？
6. **信任破绽** — 隐形质疑点？要不要主动自揭？
7. **事实准确** — 数据/履历/数字 全部对得上项目记忆？
8. **留存曲线** — 每5-15秒有钩？最强钩在30-60秒？

## 10. 当前战略主线（3个月）

### 家里开不现实商店 · 把客厅变成一家店

**为什么这是主线**：
- 一次解决多个痛点（要反馈 / 缺创新 / 总在讲道理 / 累于知识付费）
- 品牌号触顶的真正破局点（从电商品牌升级为有实体的品牌）
- 内容天然成系列 不用每天想发什么

**四周硬节奏**（避免拖进新焦虑）：
- W1 · 清一面墙 + 量尺寸 + 发"我要干一件不现实的事"
- W2 · 货架 + 展示台
- W3 · 灯光 + 背景 + 第一件商品上墙
- W4 · 第一次15分钟试播

**主线挂品牌号** · AZ号吃幕后花絮+第一人称决策视角

## 11. 异常处理

| 情况 | 处理 |
|---|---|
| 一条没爆 | 不慌 · 看24/48小时数据 · 收藏率比赞数重要 |
| 评论区有杠 | 不回 · 立场片允许争议 |
| 老粉问"啥时候上新" | 透露一点不剧透 · 制造期待 |
| 流量明显下滑 | 检查AI标记 / 检查是不是连发同主题 / 看是不是审美疲劳 |
| 突然爆了 | 立即看评论区找下一条选题 · 24小时内发跟进 |
| 没信心想停更 | 停更1-2天不会塌 · 去看评论看用户看真人 · 回来再发 |

## 12. AZ这个人 · 心理锚点（不能忘）

- 内容是副产物 · **设计是主业**
- 轻量化是本能的自我保护 · 别推翻
- **羡慕的是"生活+兴趣"型博主** · 方向已定 · 不再纠结
- 粗糙但统一 · 辨识度来自一致不来自精致
- 卖课犹豫是对的 · 不硬上
- 一个人做品牌 · 一边做一边怀疑 · 这是正常的
- 状态不对的时候 不录立场片 · 等心情回来
