## 脚本正文

**【0-8秒 · 利他钩 + 观众认领】**

如果你现在在广告公司上班，这条是讲给你听的。

不是骂你们，我以前也在里面干了十几年。我想跟老同事聊件事。

**【8-30秒 · 数据 + 共情切入】**

最近数英出了一篇文章，叫《4A公司怎么做小红书，快来抄作业》。我点进去看了一圈。

TBWA 4千粉。McCann 1千6。阳狮、蓝标、Havas Events，加起来也就几千。全球顶级4A在小红书，都是这个数。

我知道你们心里在想什么——"我们公司那个号，说实话我也看不下去。"

我也看不下去。这不是哪家做得差，是集体在这儿卡住了。

**【30-65秒 · AZ自己的故事 · 扎心不指责】**

我在4A干了十几年，管过30人的团队。

广告人的成就感长在哪？跟你说实话——长在**比稿那个会议室里**。客户的marketing总监说"就这版"那一刻，整个团队一起欢呼。我可以开心一个月。

这是我们这代广告人的集体记忆。

但我离职之后做了一个卖键帽的小品牌。第一款产品3天卖出200套——**200个陌生人，真金白银买**。

那种感觉，跟比稿赢了，完全不一样。

我才反应过来——**我过去十几年天天开心的那个"认可"，其实只是两三个人的认可**。

**【65-90秒 · 判断 · 不是人的问题】**

所以4A做不好小红书，不是不会做。是**肌肉长在别的地方**。

长在"说服两三个人"上。过brief、过创意会、过法务、过客户初审终审。每一层都在消灭意外、打磨棱角。最后出来的东西——不会错，但也不会被人记住。

这套肌肉在央视时代是顶配。

但小红书要什么？要口误、要素颜、要破音、要一个人对着镜头叹一口气。

**这两件事的难度，是反的。**

**【90-110秒 · 自反驳 · 不是人的问题是结构的问题】**

可能有人说——4A也有转型做得好的人，做个人号做得风生水起。

对，有。但那些人，全是**脱离机构之后做的**。一回到公司账号、回到那句"这条要过法务"，同一个人也做不了。

所以不是人变了，是**结构的问题**。机构做个人媒介，天然别扭。

**【110-130秒 · 具体动作 + 评论钩 + 回主理人身份】**

所以我不是站在外面说风凉话。我也是从这里出来才慢慢想明白。

如果你还在里面，别等公司号做好。**你自己先做一个**。

这周随便找个晚上，打开手机，录60秒——讲你**最近被毙掉的那个idea**。别删、别配BGM、别加字幕，就讲。

被毙掉的那些才是好东西。过了审的都不是。

**评论区告诉我你最近被毙掉的那个稿——我这几天在做下一款键帽，一条条看。**

## 这版为什么能留住人（留存曲线重判）

| 时间点 | 内容 | 留存动作 |
|---|---|---|
| 0-8秒 | "如果你在广告公司上班，这条是讲给你听的" | 目标观众认领，拒绝的人提前划走（反而利好算法） |
| 8-30秒 | 数据反差 + "我也看不下去" | 共情触发，广告人留下 |
| 30-45秒 | "管过30人团队""比稿开心一个月" | 圈内人画面感，点头 |
| 45-65秒 | "200个陌生人真金白银买"的反转 | **最强留存钩**——听到这儿的人99%会听完 |
| 65-90秒 | "肌肉长在别的地方" | 观点成立 |
| 90-110秒 | 自我反驳 | 说服力升级 |
| 110-130秒 | 具体动作+评论钩+"做下一款键帽" | 回到主理人身份，引爆评论区 |

**关键钩**：45-65秒那段反转，是全片的留存引擎。200个陌生人 vs 两三个人——这个对比一出来，后面的判断都有根了。

## 对比v2·为什么这版更对

1. **不指手画脚**——v2是"以前你们现在你们"，v3是"我当年我也是这样"
2. **有具体画面**——v2的"造神vs做人"是抽象理论，v3的"比稿会议室欢呼vs 200个陌生人买"是画面
3. **不冤枉人**——v2拿TBWA自嘲当棍子；v3整个不用这个点
4. **数据诚实**——不说"我13万"这种讨巧的话
5. **回得来**——v2结尾停在观点上（"没人再看slogan了"），v3结尾回到"我在做下一款键帽"——**把账号拉回主理人身份**
6. **有反方自驳**——"可能有人说…对，但…"，立场更高级
7. **具体动作**——"录60秒被毙的idea"——给了观众可做的事
8. **评论区自己会长**——"告诉我你最近被毙的那个稿"——这是天然的UGC触发器
