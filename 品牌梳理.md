# 不现实商店 · 品牌梳理

## 品牌洞察

你每天触碰最多的东西，恰恰是你最无感的东西。

键盘每天敲几万次，你从没认真看过它。
杯子每天举起七八回，你从不知道它长什么样。
手机壳每天摸几百次，你已经忘了它是什么颜色。

这些物件占据了你生活中最多的"触碰时间"，但你和它们之间没有任何故事。

不现实商店做的事，是给这些"最近的陌生人"一个值得在意的理由。

## 品牌定义

不现实商店是一个为手边物件注入叙事的品牌。

我们相信，你触碰最多的东西，不应该是你最无感的东西。
键帽、杯子、手机壳 — 它们每天都在你手里，但从来没有进入你的内心。
我们为这些物件写故事、造世界、做工艺，
让它们从"放在那里"变成"值得看一眼的东西"。

### 一句话定义

为你触碰最多的东西，造一个值得在意的世界。

## 品牌不是什么

- 不是文创杂货铺（不是什么都做，每个产品都有世界观门槛）
- 不是IP周边商（不依附于某个IP，而是用文学/幻想作为创作素材）
- 不是只卖好看的（好看是结果，叙事才是原因）
- 不是只讲情怀的（有专利工艺，有供应链，有清晰的商业模式）

## 品牌主张

认真不现实

重音在"认真"，不在"不现实"。

## 品牌态度

对不现实的事极其认真。

认真到每颗键帽都有一整个世界观。
认真到杯子内壁的画面申请了专利。
认真到给一套键帽写的故事比很多品牌的品牌手册还长。

主理人原话："我只是想把那些被轻易否定的想法，认认真真地做一次。不计结果，不讲效率，不急着成功。只是把一个想法，从脑子里带到手边来。"

## 目标人群

对手边物件有审美要求，但市面上的选择要么太无聊要么太幼稚的人。

共同特征：
- 花过心思布置桌面/工位
- 买过不止一副键帽或手机壳
- 对"联名"两个字已经有点疲惫，因为大部分联名只是贴个logo
- 希望自己用的东西"有点什么"，但说不清到底要什么
- 不是为了给别人看，而是给自己的日常加一层质感

一句话：想让手边的东西配得上自己花的时间的人。

## 品牌结构

世界观是横轴（故事），品类是纵轴（载体）。

### 世界观体系

- 魔法系统 / The Arcane Systems
- 文学宇宙 / The Literary Realms
- 幻想地理 / World Mapping Project
- 武道哲学 / Way of the Blade
- 万物理论 / Theory of Everything
- 大师美学 / Master Aesthetics
- 世界入口 / Entry Points
- 失真电台 / Distorted Broadcast

### 产品品类

- 键帽
- 键盘
- 桌垫
- 杯子
- 壳类（手机壳 + 耳机壳）

### 联名/限定

- 上海大剧院
- 刘慈欣漫画宇宙
- 流浪地球
- 悲惨世界

同一个世界观可以横穿多个品类，同一个品类也可以承载多个世界观。
这个交叉结构是品牌和普通"好看键帽品牌"的核心区别。

## "认真不现实"怎么用

不是口号喊一喊。体现在品牌的每一个行为里：

- 产品上的认真：每个产品都有一整个世界观，专利工艺
- 内容上的认真：品牌手册本身就像一本杂志
- 合作上的认真：不叫经销商叫"物件代理人"
- 态度上的认真：主理人自述不装酷不喊口号

市面上讲"不现实"的品牌很多，但真正把"不现实"当成一件正经事来做的，几乎没有。
