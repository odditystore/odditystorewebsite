**关于启示魔法日报（旗舰产品故事）：**
- 这是AZ第一个键帽产品，也是最成功的爆品，两年卖了近2000套

**关于AI在工作中的真实角色（这个澄清非常重要）：**
- 很多人以为AZ的产品是AI直接做的，这完全不是事实
- 真实工作流：1）AZ心里先有想法和方向（brief思维，广告训练出来的）→ 2）写提示词让AI生成概念图（提示词是AZ写的，方向是AZ定的）→ 3）AZ筛选哪张接近自己想要的 → 4）发到平台测试市场反应 → 5）所有实际产品设计全部手工重新做（AI草稿不能直接给工厂，工艺达不到、精度不够、细节全是错的）
- AI真正的价值：把市场验证的成本压到几乎为零，不是替你做产品

**关于完美主义的教训：**
- 好几个产品延期上线，原因都一样——执念太重，非要100%还原脑子里的画面
- 大部分用户根本不知道AZ脑子里那个"完美版本"长什么样
- 产品做到80分可能花20天，从80到100可能三个月都到不了
- 现实限制：开模具两万起，如果死磕完美最后卖不出去，商业循环就断了
- 他确实能花1000块完美还原一个产品，但只能卖900，这个生意就不成立
- 教训：先跑通再完美，别先完美再跑通
- 可以用更轻的方式验证：3D打印+手工涂色快速测试，有流量再开模具

**关于品牌理念：**
- "不现实商店"的核心是把不现实的、不太可能做出来的东西，用现有工艺尽可能做出来，真正让它到达用户手里，而不是永远只是一张图
- 审美是护城河。一定有人喜欢也有人不喜欢，只需要服务那一小撮人。中国市场够大，一小撮人就够活得很好

**关于市场测试和选品经历：**
- 2023年起号时做过手柄、雨伞、机箱、耳麦、电竞椅偏男性向产品，全没成功。可能是小红书平台本身的女性属性原因
- 最后键帽跑出来了，不是AZ主动选的，是市场帮他选的
- 方法：用AI出概念图发到平台看数据反应，数据好的深化，不好的换

**关于一个人做品牌的核心方法论：**
- 时间复利：卖产品而不是卖时间，一个人买和一千个人买付出时间不变
- 产品不限于实物：手工材料包、知识付费课程、设计模板都算产品
- 轻测试：用AI概念图快速验证市场，一天测很多组，试错成本几乎为零
- 不同品类的测试方式：实体→AI概念图看反应；手工→拍制作视频看有没有人问"能买吗"；知识付费→免费发核心内容看有没有人追完整版；模板→放预览图看有没有人要源文件
- 审美是护城河：不只是视觉审美，做手工的是对材质工艺的理解，做知识产品的是对内容结构的把控
- 找不到设计师可以在小红书上找，成本不高，关键是把想法说清楚

## 破局方向：不现实商店实体化（家里开店+过程直播）

**定性**：这不是选题，是**战略方向**。AZ双号体系的第三条腿，可能是破品牌号流量天花板的关键动作。

### 为什么这个方向对（多层论证）

**1 · 赛道红利**
- "在家做XX"（咖啡店/画廊/工作室/美术馆）是过去两年小红书最火的母题之一
- "在家真开一家商店 · 真迎客 · 真交易 · 真做出产品"——**这个细分赛道几乎没人做**
- AZ的差异化不是"在家"本身，是**"把一个品牌物理化"**——这是独一份

**2 · 一次性解决AZ当前多个痛点**
- "内容总在讲道理" → 装修和开店是具体的手的事
- "需要正向反馈" → 每天进度肉眼可见 / 直播互动比评论快10倍
- "创新少了" → 这是最大的创新 · 线下×线上的罕见组合
- "想放弃知识付费" → 不用纠结 · **店本身就是最好的课堂**
- "输出疲劳" → 回到手的工作 · 治疗创作者抽象化焦虑

**3 · 战略维度升级**
- 对品牌号（不现实商店，13万粉）：从"电商品牌"升级为"**有实体的品牌**"——认知维度质变，这是流量天花板的真正破局点
- 对AZ号（156粉，起号）：从"讲道理的人"变成"做东西的人"——行动片比观点片更适合AZ号起步阶段
- 对商业：打开了"面基体验/驻店活动/限定款"等新变现路径

**核心判断**：这件事**主线在品牌号**，AZ号吃幕后。反过来最大化两个号的差异化。

### 真正的爆点是"家里"不是"店"

- "开店"是结果，观众对别人开店不共情
- "**把客厅变成一家不存在的店**"是过程，观众会追着看
- 叙事锚：**"一个人，不租店，在自己家，把想了很久的店做出来"**——这个叙事几乎通杀所有"一个人做品牌"的共情点

### 延伸产品想象力（不只是陈列）

1. **客厅限定款**：只在这个店里能买到的特别版——制造稀缺
2. **面基体验**：核心粉丝可约时间到家——最高等级用户的gift
3. **驻店创作**：每月邀请一位独立设计师客厅驻店一周——社区+联动
4. **手作直播**：AZ当场做键帽/丝网印刷/组装/试错——高手动手是刚需内容（参考丹麦木匠、日本陶艺直播模式）
5. **店长日志**：每天一条短图文记录店内发生的事
6. **"不营业"的仪式**：某天开门某天关门，非7*24，稀缺感

### 直播具体打法（基于平台差异）

- **小红书直播**：陪伴型，每周一次"店铺进度展示播"30-60分钟，不卖货，聊装修聊产品聊灵感
- **抖音直播**：转化型，暂不做——AZ主阵地不在这里
- **视频号直播**：可做，但优先级低
- **重要原则**：**不要一开始就日播**——会消耗能量反噬内容质量。从每周1次试播起步

### 最小可行版本（四周时间线 · 硬节奏）

**四周硬节奏的意义**：设硬时间线是为了**每周都有看得见的进展**——治愈AZ当前"没反馈"的焦虑。如果拖半年才直播，会再次陷入焦虑循环。

- **第1周**：决定+清空一面墙+拉尺寸 → 发"我要干一件不现实的事"
- **第2周**：第一组货架+展示台 → 发开箱+安装视频
- **第3周**：灯光+背景+第一件商品摆上 → 发第一张成品照片
- **第4周**：第一次试播15分钟 → 不求完美，把"开门"这个动作做出来

**预算**：两三千起步，不要一次全套装修。

### 风险和trade-off（AZ可能没想到）

1. **生活边界**：客厅被占，家人/生活场景暴露——事先和家里人沟通，明确哪些不入镜
2. **换场景成本**：一旦"家里开店"成为品牌视觉锚点，未来换场景（搬家/扩大）成本高——设计时考虑"可拆可移"
3. **直播能量反噬**：长期直播消耗心理能量，会影响内容构思——**严格控制频率**，每周1次上限
4. **稀缺感管理**：如果随时直播+随时开放，会失去"不现实"的稀缺感——建议做"营业时间"制，比便利店更神秘
5. **产品量级匹配**：现有产品够不够撑一个"像样的店"的感觉——先盘点SKU，不够就先做几个专为店做的视觉件

### 与"一个人做品牌"叙事的契合

这条方向和AZ现有的"一个人做品牌"系列不是并行，是**升级**——
- 系列1（卖产品而不是卖时间）→ 2 现在可以加一条：**做空间而不是做号**
- 系列2（AI工作流）→ 可以加一条：**物理空间是AI的反面**，两者结合是完整的品牌
- 系列3（80分上线）→ 完美对应："店也是先开门再完美"

这件事做好了，不仅解决流量和心理问题，还会把"一个人做品牌"这个系列从"方法论"升级成"行动示范"。

### 公开/内部分层

- **可公开的**：全过程记录、装修细节、试错、直播互动
- **建议内部**：具体预算数字（避免复制作弊 · 保留神秘感）、面基的具体机制（先跑通再讲）
- **绝对不公开**：这是品牌号流量触顶的破局行动（战略意图不暴露）

1. **"图里有我"（数英文章反论）** → 暂缓。有了家里开店这个方向，"图里有我"的迫切性降低——行动片比观点片更适合AZ当前阶段

## 二、双号定位与调性

产品本质上是情绪商品，不是功能商品。

### 核心战略（2026.4更新 · 重要）

**AZ号的真正使命不是"做第二个博主"，是给品牌号补"人的那面"**。

品牌号（不现实商店）的基因是"AI+概念+成品+世界观"——精致、偏冷、偏机器感。这是它13万粉的立身本钱，**也是它流量触顶的部分原因——很多人抗拒AI感**。AZ一直想"证明一些什么 / 要有活人感"——这就是AZ号的核心使命。

所以两个号的分工不是"两个独立博主"，是**互补的一对**：

| | AZ的不现实商店 | 不现实商店 |
|---|---|---|
| **使命** | 证明"**活人感**" · 补品牌号的人的那面 | 品牌美学 · 成品和世界观的呈现 |
| **定位** | 生活+创作过程型博主 | 概念商品号 |
| **调性** | 松弛、真实、自嘲、有手有脸有犹豫有失败 | 认真中二、浪漫、仪式感、美学统一 |
| **内容** | 过程、手、脸、声音、失败、叹气、日常 | 产品、世界观、纯视觉 |
| **气质** | 热、近、活人、粗糙 | 冷、远、作品、精致 |
| **核心KPI** | 不是粉丝数，是**活人感的建立** | 粉丝数+带货 |

**关键判断：AZ号不是"创业博主讲方法论"的号，也不是"教程博主"。** AZ欣赏中gone的课程但自己本能拒绝做课程博主，羡慕的是"围绕生活与兴趣展开"的博主（养叶子+手工+探险+经营家那种）——这是AZ号的内在模板。

### AZ号类型定位（2026.4确认）

**类型：生活 + 创作过程型博主**

- 不做：系统教程、大课、种草开箱、讲方法论
- 做：真实记录、过程、手作、家、设计日常、客厅开店

吸引的人群：
- 做独立品牌的人
- 喜欢一个人折腾的人
- 审美向的消费者
- 对"如何过一种有创作感的生活"感兴趣的人

**不吸引**（也不该勉强吸引）：
- 想快速学方法论的人（他们该去找中gone）
- 想快速买东西的人（他们该去找麓叔）

### 品牌基因要融进主理人号里

- 自嘲和中二不矛盾："所有人说没人买，确实没怎么卖，但李白也没靠写诗赚钱"
- 松弛和认真不矛盾：一边说"这不现实"，一边把键帽认真撒成一首诗
- 搞笑和浪漫不矛盾：用最不正经的方式讲最认真的事

AZ号和不现实商店号是同一个人的两面，不是两个人格。

## 二·之二、AZ号美学系统（2026.4新增）

**核心悖论**：AZ美术出身对调色敏感 → 想把自媒体拍得精致 → 但精致化=AI感方向，和"活人感"使命反向。

**解法：产品拍摄和AZ号拍摄彻底分开**，用不同标准。

| | 产品图（工作） | AZ号内容（生活） |
|---|---|---|
| 标准 | 高质量 · 有美感 · 调色讲究 | 真实 · 有人感 · 不修图 |
| 设备 | 相机+灯+调色软件 | 手机 · 原片 |
| 时间投入 | 投入 · 值得花时间 | 随手 · 不超过10分钟 |
| 意图 | 卖货 / 品牌视觉 | 让人相信背后是个活人 |
| 风格参考 | 不现实商店的统一美学 | 自己日常的真实 |

**AZ号拍摄的"有辨识度的不完美美学"三件套**（固定 · 不再决策）：

1. **光**：永远自然光。窗边、阳台、户外。不开大灯。没自然光就别拍
2. **色**：一个轻预设（建议偏暖偏旧）。10秒套完不再动
3. **构图**：二选一——居中+留白 或 45°俯拍

**禁用清单**（AZ号绝不用）：
- ❌ 三脚架、稳定器
- ❌ 调色（除了那个一键预设）
- ❌ 剪辑花字 / 动效
- ❌ BGM（干声就是活人感）
- ❌ 精修 / 磨皮 / 加滤镜

**参照物**：Peter McKinnon车库记录片 / 日本UP主PORI / "一条"早期手持跟拍——共同点是**不精致但极其统一**。辨识度来自一致，不来自精致。

**节奏建议**：
- 用三周时间AZ号只用一套风格拍
- 拍得不好也别调整，忍住
- 三周后你会发现你不用"想怎么拍"了，你只想**拍什么**
- 累的就不是拍摄了，是生活本身——而生活本来就累 不是多出来的压力

**活人感的技术信号清单**（越多越活）：
- 有人脸出现
- 有手的工作镜头
- 能听到真实的说话声音（包括停顿、犹豫、口误）
- 场景有生活痕迹（咖啡渍、凌乱的桌面、家人的背影）
- 记录失败和返工
- 表达累、烦、笑、叹气等情绪
- 原生光而不是布光

## 三、内容策略

### 三个发布前必问
1. 别人看完能学到什么？
2. 别人看完能马上做什么？
3. 这条跟AZ的身份有什么关系？

### 内容结构

- 前30%：真实场景+故事引入（从观众的痛点切入，不是"我怎样"，而是"你是不是也这样"）
- 中间50%：实操干货
- 后20%：观点或互动引导

### 语气硬性要求（2026.4月起执行）
- **松弛，像跟朋友喝酒聊天，不是在做分享**
- 少说"你应该"，多说"我当时"——经验从故事里出来，不从道理里出来
- 多自嘲自黑，讲自己的蠢事，不当导师
- 别替观众总结道理，讲完故事让他们自己悟
- "我也不知道对不对，反正我是这么干的" > "更聪明的方式是"
- 可以笑、摇头、叹气，让情绪跟着内容走，别全程一个调

### 关键判断
- 不做纯个人经历分享，每条都要有利他角度
- 不做纯AI测评，AI内容从"主理人用AI提效"角度切
- 7成轻量日常+3成深度内容，日常工作本身就是素材
- 覆盖不同品类用户，不只讲键帽
- 系列内容要有联动引导
- 减少纯图文，以口播视频为主
- 钩子要跟正文有逻辑关联，不能为抓人硬凑悬念
