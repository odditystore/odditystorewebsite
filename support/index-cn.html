常见问题

如何在多台设备之间切换蓝牙？
按住 FN 键，同时按 Q、W 或 E，分别切换蓝牙1、2、3频道。如需重新配对某个频道，长按对应的 FN+Q/W/E 约3秒，指示灯快速闪烁后即进入配对模式，在设备端搜索键盘名称连接。
如何查看电量？
按 FN + Space（大文豪75）/ FN + Space 即可查询。RGB指示灯会短暂闪烁：绿色表示电量充足（60%以上），黄色表示中等电量（20-60%），红色表示电量偏低（低于20%）。
如何切换 Windows 和 Mac 模式？
大文豪75：按 FN+M 切换。大文豪100：按 FN+A 进入WIN模式，FN+S 进入MAC模式。Mac模式下Win键与Alt键互换，映射为Command和Option。
键盘连接不上怎么办？
大文豪75：长按 FN + 左Shift + Backspace 约3秒恢复出厂设置，然后重新配对。大文豪100：长按 FN + Del 约3秒恢复出厂设置。注意确认当前频道没有被其他设备占用。
不现实商店键帽适配这款键盘吗？
适配。所有不现实商店键帽均采用MX轴兼容的轴柱设计，完美兼容大文豪75和大文豪100键盘套件。各款键帽产品页有具体配列兼容说明。
如何关闭RGB背光节省电量？
大文豪75：按 FN + Enter 开关RGB。大文豪100：按 FN + PrtSc 关闭或打开背光。关闭背光可显著延长无线续航。
