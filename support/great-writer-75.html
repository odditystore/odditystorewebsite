Welcome to the Great Writer 75. This manual covers connection setup, key functions, RGB control, and battery management. Keep this document for future reference.

02

Key Layout

The Great Writer 75 features a compact 75% layout with 84 keys, retaining the function row and arrow cluster while reducing overall footprint. Compatible with standard MX-mount keycap sets.

03

Connection Guide

Great Writer 75 supports three wireless modes and wired USB-C. Use FNQ/W/E to switch Bluetooth channels, FNR for 2.4G, and FNT for USB.

01
Bluetooth
Press FN+QFN+W or FN+E to enter pairing mode on channel 1, 2, or 3.

Hold 3 seconds until LED blinks rapidly. Select "Great Writer 75" on your device.
02
2.4G Wireless
Insert the USB dongle. Press FN+R to activate.

The keyboard auto-pairs with the included receiver. No additional setup required.
03
USB-C Wired
Connect via USB-C cable. Press FN+T to switch to wired mode.

Wired mode also charges the battery simultaneously.
WIN / MAC Mode

Press FN+M to toggle between Windows and Mac key mapping. On Mac, the Win key functions as Command and Alt as Option. RGB indicator briefly flashes to confirm the switch.

NKRO: Press FN+N to toggle full N-key rollover. Recommended for gaming or fast typing. Note: NKRO is only available in USB-C mode.

05

RGB Lighting

Use FN + Arrow keys to cycle modes and adjust brightness. FN + A/S/D/F/G/H for fine-tune control of hue, saturation, and animation speed.

Cycle Mode
FN + ← / → to switch between lighting modes
Brightness
FN + ↑ / ↓ to increase or decrease brightness
Hue
FN + A (decrease) / S (increase) to shift color
Saturation
FN + D (decrease) / F (increase) for color intensity
Animation Speed
FN + G (slower) / H (faster) for effect speed
Toggle On / Off
FN + Enter to enable or disable RGB entirely

Note: RGB settings are saved automatically. Your preferred mode and brightness persist after power cycling.

06

Battery & Power

Battery Indicator Press FN + Space — RGB briefly flashes to indicate charge level:
● Green = above 60%   ● Yellow = 20–60%   ● Red = below 20%
Charging Connect USB-C cable. Keyboard continues to function while charging. Charge indicator light turns off when fully charged.
Power Saving Keyboard enters sleep mode after ~3 minutes of inactivity. Press any key to wake.
Reset Hold FN + Left Shift + Backspace for 3 seconds to restore factory defaults. All Bluetooth pairing data will be cleared.
