## 脚本正文

**【0-5秒 · 钩子】**

我们为什么会相信好莱坞那些科幻奇幻大片？不只是因为特效。

**【6-12秒 · 道具的秘密】**

电影里那些道具，瓶子上的标签、桌上的文件，镜头扫过去就一秒。但道具组还是把每一行字都写上去了。因为只有创作者自己相信这个世界是真的，观众才会相信。

**【13-25秒 · 草药学展示】**

我做了一套键帽叫魔法草药学。包装是一本草药学课本，打开是三层收纳盒。说明书是一张草药档案，桌垫是植物标本研究台。130颗PBT键帽加14颗金属喷涂，每颗都是一种魔法植物。

你看到这些你是不是觉得——这人一定是个完美主义者。

**【26-40秒 · 反转】**

但我跟你说我第一个产品是什么样的。

启示魔法日报。尺寸做错了，原厂比例5:6我做成了正方形。颜色校不对，换了两家工厂。打样都没定稿，我就挂到小红书上卖了。

3天200套。

如果我当时等做到现在这个水平再卖，这个品牌就不会存在。因为我之前做过一个播客，花了几周做到完美才发，发出去没人听，直接死了。

完美主义害死了多少人的第一个产品，你不知道。

**【41-50秒 · 收尾】**

你现在看到的这些细节，不是因为我一开始就能做到。是因为我先把一个不完美的东西卖出去了，有了反馈、有了收入、有了成长，才慢慢做到这个程度。

先发出去，再变好。顺序反了，什么都不会有。
